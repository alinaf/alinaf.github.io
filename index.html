Here are some game prototypes made using Phaser 3! Phaser Labs was my go-to resource.

grabble

play code

Grabble is a 2-player word game in which players make words from a communal tile bank, and can steal each other's words by adding to them and/or rearranging them. This is my favorite board game, and I play with Bananagrams tiles. Grabble uses Heroku + Node.js!

To play, press the start button after you and a friend have both launched the site.
